GNU/Linux

Що таке GNU/Linux?

GNU/Linux — це сімейство операційних систем на основі ядра Linux та інструментів проекту GNU. Це вільні і відкриті ОС, що дозволяють користувачам змінювати та поширювати програмне забезпечення. Вони часто використовуються на серверах, вбудованих системах та особистих комп’ютерах.

Безкоштовність

Більшість дистрибутивів Linux є абсолютно безкоштовними. Це приваблює людей, які не хочуть платити за комерційні операційні системи, як-от Windows чи macOS..

Спільнота та підтримка

Linux має величезну та активну спільноту, яка допомагає новачкам та досвідченим користувачам через форуми, документацію та керівництва. Є багато ресурсів, де можна отримати підтримку чи вирішити проблему.

Командний рядок

Для багатьох користувачів можливість керування системою через потужний командний рядок є великою перевагою. Це дозволяє автоматизувати завдання, керувати серверами та налаштовувати систему ефективніше..

Підтримка старого обладнання

Linux менш вимогливий до ресурсів і може працювати на старішому обладнанні. Це дає змогу продовжити термін служби старих комп’ютерів.

Стабільність та надійність

Linux славиться своєю стабільністю, особливо на серверах. Багато компаній та організацій використовують Linux на своїх серверних платформах, оскільки система рідко вимагає перезавантаження та може працювати безперервно протягом тривалого часу.

Культура вільного програмного забезпечення

Багато користувачів підтримують ідеологію вільного та відкритого програмного забезпечення (Free and Open Source Software), що сприяє розвитку GNU/Linux і програм для нього.

Основні дистрибутиви

Найбільш попурялні серед користувачів дистибутиви GNU/Linux

Ubuntu

Один із найвідоміших дистрибутивів. Орієнтований на зручність використання і підходить для новачків. Має активну спільноту та регулярні оновлення.

Debian

Стабільний і надійний дистрибутив, який є основою для багатьох інших, таких як Ubuntu. Використовується на серверах і для особистого використання.

Read More

Fedora

Розробляється за підтримки компанії Red Hat. Зосереджується на сучасних технологіях і часто служить випробувальним майданчиком для нових рішень, які пізніше з'являються в Red Hat Enterprise Linux.

Read More

Arch Linux

Призначений для просунутих користувачів. Вимагає ручного налаштування, але пропонує високу гнучкість і контроль над системою. Має активне співтовариство і систему оновлень "rolling release".

Приклади команд роботи в Linux CLI

pwd, echo, ls, cd, touch, mkdir, cp, mv, rm, history

pwd (Print Working Directory)

Відображає поточну директорію, в якій ви зараз працюєте.

pwd

Приклад виводу: /home/user

echo

Виводить текст у терміналі або записує текст до файлу.

echo "Hello, Linux!"

Приклад виводу: Hello, Linux!

ls (List)

Виводить список файлів і директорій в поточній директорії.

ls

Приклад виводу: file1.txt file2.txt folder

Для детальнішого виводу з правами доступу, датами, розмірами:

ls -l

cd (Change Directory)

Переміщення між директоріями.

cd /home/user/documents

Повернення на рівень вище:

cd ..

touch

Створює новий порожній файл або змінює час модифікації файлу.

touch newfile.txt

mkdir (Make Directory)

Створює нову директорію.

mkdir newfolder

Створення вкладених директорій:

mkdir -p folder/subfolder

cp (Copy)

Копіює файли або директорії.

cp file.txt /home/user/backup/

mv (Move)

Переміщення файлу в іншу директорію:

mv file.txt /home/user/documents/

Переміщує або перейменовує файли та директорії.

mv file.txt /home/user/documents/
